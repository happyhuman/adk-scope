# Project Context & Instructions

## Code Formatting

When generating or modifying Python code for this project, you MUST adhere to the following standards:

1.  **Line Length**: All lines must be limited to a maximum of **80 characters**.
2.  **Standard Conventions**: Follow PEP 8 style guidelines for naming, indentation (4 spaces), and structure.
3.  **Imports**: Sort imports according to standard Python practices (standard library, third-party, local).
4.  **Type Annotation**: Use type annotations for function and method arguments and return values where applicable.
5.  **Docstrings**: All functions and methods must have docstrings.
